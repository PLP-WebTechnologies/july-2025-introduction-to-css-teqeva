What is CSS?

CSS (Cascading Style Sheets) gives life to your HTML. With colors, spacing, and layout control, CSS makes web pages not just readable—but beautiful.

The Box Model

Every element on a page is a box. The Box Model explains how these boxes work:

  • Content – the core text or images
  • Padding – space inside the box
  • Border – the outline around the box
  • Margin – space outside the box

Why it Matters

Understanding the Box Model is essential for layout design. It’s the foundation of spacing, alignment, and responsive design on the web.
